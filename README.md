## Pixel Perfect Portfolio

This repository contains a React + TypeScript + Vite frontend for a pixel‑perfect portfolio site.

All application code and tooling live inside the `frontend` folder.

### Getting started

```sh
cd frontend
npm install
npm run dev
```

Then open the URL that Vite prints in your terminal (usually `http://localhost:5173`).

### Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
